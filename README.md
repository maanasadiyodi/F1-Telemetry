# F1 Telemetry Analytics

Professional-grade Formula 1 telemetry analysis tool built with FastF1 and Flask.

## Features

- 📊 Speed Analysis
- 🎬 Live Replay with pedal inputs
- ⚔️ Driver Comparison
- ⏱️ Lap Times Analysis
- 🏁 Race Positions
- 🔧 Tire Strategy
- 🗺️ Track Map

## Tech Stack

- Backend: Python, Flask, FastF1
- Frontend: Vanilla JavaScript, Chart.js
- Data: Formula 1 Official API

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
